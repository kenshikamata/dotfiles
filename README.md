# About
* this is my dotfiles

# How to install
* `$ cd ~/dotfiles/`
* `$ ./install.sh -option`
* then my dotfiles will be installed.
## options
* `-v`
	* only vim settings will install
* `-z`
	* only zsh setting will install
* `-t`
	* tmux setting will install
* `-a`
	* vim, zsh, tmux, all setting will install
